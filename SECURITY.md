# Security Policy

## Supported Versions

For now only the latest major version is supported

| Version | Supported          |
| ------- | ------------------ |
| 6.x.x   | :white_check_mark: |
| <= 5.0.x| :x:                |

## Reporting a Vulnerability

You can report any problem you encounter in https://github.com/destbg/SQLite.Framework/issues
